# Public Folder

This folder contains static assets that are served directly by Next.js.

## Logo Image

**Place your logo image file here with the name: `logo.png`**

### Supported formats:
- PNG (recommended)
- JPG/JPEG
- SVG
- WebP

### File location:
```
public/logo.png
```

The logo will automatically appear in the top left corner of the landing page.

### Alternative file names:
If your logo has a different name or format, you can update the file path in `app/page.tsx`:
- Change `src="/logo.png"` to match your file name
- For example: `src="/logo.jpg"` or `src="/company-logo.svg"`
